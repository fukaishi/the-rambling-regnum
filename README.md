# The Rambling Regnum: The Never-Ending Prologue

勇者以外の登場人物の話が、ただひたすら長いことだけが特徴のドット絵風RPG

## ゲームの特徴

- 🏠 勇者が朝起きて、母親の話が長い
- 🏰 王様に話しに行って、王様の話が長い
- 👹 ラスボス討伐に行って、ラスボスの話が長い
- ⚔️ 戦闘中も何かするたびに話が長い
- 🎮 3ターンで倒せるが、倒れるまで話が長い
- 🗺️ 自宅→お城→ラスボスの場所がそれぞれ目の前

## 操作方法

- **矢印キー (↑↓←→)**: 勇者を移動
- **Enterキー**:
  - 起きる（ゲーム開始時）
  - NPCと会話する（隣接時）
  - 会話を進める
  - バトル中に攻撃する

## ゲームの流れ

1. Enterキーで勇者が起きる
2. 矢印キーで移動して母親に近づき、Enterで会話開始
3. Enterキーで会話を進める（17メッセージ）
4. 上に移動して王様に近づき、Enterで会話開始
5. 会話を進める（20メッセージ）
6. さらに上に移動して魔王に近づき、Enterで会話開始
7. 会話後バトル開始（18メッセージ）
8. Enterキーで攻撃 → 魔王の長話 を3ターン繰り返す
9. エンディング

## 技術スタック

- React 18
- Vite
- CSS (ドット絵スタイル)
