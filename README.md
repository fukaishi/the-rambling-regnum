# The Rambling Regnum: The Never-Ending Prologue

勇者以外の登場人物の話が、ただひたすら長いことだけが特徴のドット絵風RPG

## ゲームの特徴

- 🏠 勇者が朝起きて、母親の話が長い
- 🏰 王様に話しに行って、王様の話が長い
- 👹 ラスボス討伐に行って、ラスボスの話が長い
- ⚔️ 戦闘中も何かするたびに話が長い
- 🎮 3ターンで倒せるが、倒れるまで話が長い
- 🗺️ 自宅→お城→ラスボスの場所がそれぞれ目の前

## 技術スタック

- React 18
- Vite
- CSS (ドット絵スタイル)

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## デプロイ

mainブランチまたはclaude/で始まるブランチにpushすると、自動的にGitHub Pagesにデプロイされます。
